# Features

## GPU Monitoring
- Real-time utilization tracking
- Memory usage monitoring
- Temperature monitoring
- Multi-GPU support

## Kernel Profiling
- Per-kernel timing
- Launch configuration analysis
- Occupancy estimation
- Bottleneck detection

## Memory Profiling
- Allocation tracking
- Leak detection
- Peak usage monitoring
- Lifetime analysis

## Bandwidth Analysis
- Host to Device transfer metrics
- Bandwidth benchmarking
- Transfer pattern analysis

## Export Formats
- Chrome Trace (for chrome://tracing)
- JSON (for custom analysis)
- CSV (for Excel/Python)
